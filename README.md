# Sarthak Ghimire Portfolio

This repository contains the cinematic portfolio website for Sarthak Ghimire.

## Files

- `portfolio (1).html` — Main portfolio page.
- `Screenshot 2026-05-05 141943.png` — Featured photography image added to the portfolio.
- `host-local.bat` — Local web server launcher.

## Local hosting

Run `host-local.bat` and open `http://localhost:8000/portfolio%20(1).html` in your browser.
